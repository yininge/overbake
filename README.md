# overbake
A game to make perfect pizza. Hit the start button, then and hit stop baking button when the pizza is ready. 
This is an example app helps me to explore reactiveswift and reactivecocoa.

# Getting started
This app uses [CocoaPods](https://guides.cocoapods.org/using/getting-started.html#getting-started) to manage dependencies. 
- Clone the project repository
- Run `pod install`
- Open the .xcworkspace file
